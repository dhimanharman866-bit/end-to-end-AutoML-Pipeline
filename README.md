# End-to-End AutoML Pipeline

This project builds an automated machine learning pipeline where users can upload a dataset and train multiple models automatically.

## Features

- Dataset upload
- Target column selection
- Automatic model training
- Best model selection
- Streamlit interface

## Tech Stack

- Python
- FastAPI
- Scikit-learn
- Streamlit

## Run Backend

uvicorn main:app --reload

## Run Frontend

streamlit run app.py
